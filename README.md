# Pandorym scoop-repo


## Installing Scoop
In a PowerShell command console, run:
```
iex (new-object net.webclient).downloadstring('https://get.scoop.sh')
```

## Add bucket
```
scoop bucket add Pandorym https://github.com/Pandorym/scoop-repo.git
```
